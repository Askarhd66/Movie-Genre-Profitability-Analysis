<h1>🎬 Movie Genre Profitability Analysis for SussexBudgetProductions</h1>

<h2>🎯 Project Overview</h2>
SussexBudgetProductions recently experienced major financial losses with a comedy-action-thriller film and is now shifting focus toward either Romance or Horror.

This project uses IMDb data to determine:

- <b>Which genre is more profitable</b>
- <b>Which director delivers the highest‑rated Romance films</b>
- <b>Which lead actor offers the best combination of ratings and audience appeal</b>

The analysis includes **data cleaning, feature engineering, exploratory analysis, and hypothesis testing** to provide data‑driven recommendations for the next film production.

<h2>🛠 Languages, Libraries & Tools</h2>
- Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy (T‑tests)), Jupyter Notebook

<h2>📊 Datasets Used</h2>
- IMDB Dataset (The dataset contains 5,043 movies with 26 features, including IMDb ratings, Genres, Budget and gross revenue, Director and actor information, Review counts)

<h2>Key Stats:</h2>

- <b>Average gross revenue: £48.5M (max £760M)</b>
- <b>Average IMDb rating: 6.44</b>
- <b>Most common genre: Drama (236 films)</b>

<h2>🎯 Final Recommendation</h2>

SussexBudgetProductions should produce a **Romance film** directed by **Ridley Scott** and starring **Ryan Reynolds**.

This combination maximises audience appeal, IMDb rating potential, and commercial visibility.
